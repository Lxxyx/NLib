## 介绍
用于查询图书是否能借阅。一次性查询多本图书，省时省力。
## 运行方式
```
git clone git@github.com:Lxxyx/NLib.git
cd NLib
npm install
```

将要查找的图书，放入`data/list.json`中即可。
然后
```
cd dist
npm install
gulp
```
## 开发

```
// 安装完成后

```
